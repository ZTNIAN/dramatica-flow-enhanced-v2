# 优秀场景描写示例：雨夜

## 原文示例

雨下得太久了。

久到墙壁都在渗水，一滴一滴，像是房子在哭。

陈默站在窗前，看着窗外那条被雨水泡发的街道。路灯昏黄，像快要咽气的病人，忽明忽暗地撑着眼皮。

已经是凌晨两点十七分。

这个时间，不该有人敲门。

但门响了。

咚。

不是那种急促的砸门声，也不是礼貌的轻叩。而是——一下。很重。很稳。像是一个知道一定会有人来开门的人。

---

## 技巧分析

### 1. 五感描写
- **视觉**："墙壁渗水"、"路灯昏黄"、"忽明忽暗"
- **听觉**："一滴一滴"、"咚"
- **触觉**："冷风灌进来"
- **心理感受**："像是房子在哭"（拟人化）

### 2. 节奏控制
- 开头短句营造紧张感
- 长短句交错："雨下得太久了。（短）久到墙壁都在渗水...（长）"
- 关键声音单独成段："咚。"

### 3. 悬念设置
- 时间："凌晨两点十七分"（不寻常的时间）
- 反常："不该有人敲门，但门响了"
- 细节：敲门声"很重、很稳"（说明来人很确定）

### 4. 去AI味特征
- ❌ 无"首先/其次/最后"
- ✅ 口语化："这个时间，不该有人敲门"
- ✅ 有瑕疵：口语重复、不完美逻辑
- ✅ 情感真实："像快要咽气的病人"（主观感受）

---

## 可借鉴的结构

```
环境铺垫（天气/时间）
    ↓
异常现象（反常的声音/景象）
    ↓
人物反应（心理活动+动作）
    ↓
悬念升级（揭示更多信息）
```

## 练习建议

模仿这个结构，写一个类似的场景：
1. 选择一个特定时间（深夜/黎明/黄昏）
2. 描写环境（至少2种感官）
3. 设置一个异常现象
4. 展示人物的心理反应
